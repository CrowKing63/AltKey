# AltKey 아이콘 리디자인 시안

AltKey / AltKey.Tools 아이콘 방향을 빠르게 비교하기 위한 PNG 시안입니다.
원본 아이콘은 유지하고, 아래 파일만 별도로 생성합니다.

## 생성 파일

- `docs/assets/icon-concepts/altkey-icon-concept-a-edit.png`
- `docs/assets/icon-concepts/altkey-icon-concept-ga-korean.png`
- `docs/assets/icon-concepts/altkey-icon-concept-fluent-hybrid.png`
- `docs/assets/icon-concepts/altkey-icon-concepts-board.png`

## 시안 의도

### 1. A + Pencil
- 기존 아이콘 인지 요소를 가장 많이 유지합니다.
- 툴즈/편집기 성격은 우하단 연필 배지로만 추가합니다.

### 2. Korean 'ga'
- 한국어 전용 앱이라는 점을 아이콘만 보고도 더 바로 읽히게 합니다.
- 기존 순환 전환 느낌은 약하게만 남깁니다.

### 3. Fluent Hybrid
- 문자 비중을 줄이고, 창/패널/도구 느낌을 단순 형상으로 재구성합니다.
- 추후 메인 앱 / 툴즈 앱 패밀리 룩으로 나누기 쉬운 방향입니다.

## 검토 포인트

- 32px / 16px 축소 시에도 주 심볼이 읽히는지
- 밝은 제목줄과 어두운 작업 표시줄에서 모두 대비가 충분한지
- 메인 앱과 툴즈 앱을 나중에 파생형으로 나눌 때 확장하기 쉬운지
