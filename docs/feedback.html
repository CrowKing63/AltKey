사용 중 불편했던 점을 바로 알려주세요.

AltKey는 한국어 입력과 접근성을 함께 다루는 도구라서, 실제 사용 중 느낀 문제 설명이 가장 중요합니다. 재현 순서나 사용 환경을 같이 남겨주시면 원인 파악이 빨라집니다.

  • 버그 제보: 어떤 화면에서 무엇을 했을 때 문제가 생겼는지
  • 개선 제안: 어떤 흐름이 불편했고 무엇이 달라지면 좋은지
  • 접근성 의견: 읽기, 포커스 이동, 조작 난이도 관련 불편 사항
