Features

필요한 기능은 모두 담았습니다

접근성부터 생산성 자동화까지, AltKey 하나로 해결하세요.

🔒

Sticky Keys

Shift·Ctrl·Alt·Win을 한 번 탭하면 일회성 고정, 두 번 탭하면 영구 잠금. 한 손 조작이나 접근성 환경에 최적화되어 있습니다.

⏱

체류 클릭 (Dwell Click)

키 위에 마우스를 올려놓으면 설정한 시간 후 자동 클릭. 물리적 클릭이 어려운 사용자를 위한 접근성 기능입니다.

🎨

JSON 레이아웃

원하는 키 배치를 JSON으로 정의하고, GUI 레이아웃 에디터로 쉽게 편집하세요. 한글 음운 이중 레이블도 완벽 지원합니다.

🔄

앱별 레이아웃 프로필

게임, 크리에이티브 툴, 오피스 앱마다 레이아웃을 자동으로 전환합니다. WinEventHook 기반으로 포그라운드 앱을 감지합니다.

🪟

Acrylic 블러 UI

Windows 11 Acrylic 블러 효과와 다크/라이트/시스템 테마를 지원합니다. 미사용 시 자동으로 반투명 상태로 전환됩니다.

🧲

화면 가장자리 이동

헤더 중앙의 ← ↑ ↓ → 버튼 한 번으로 키보드를 화면 끝에 즉시 붙입니다. 자주 쓰는 위치에 빠르게 고정하세요.

😊

이모지 패널

이모지 패널로 자주 쓰는 이모지를 빠르게 입력하세요. 유니코드 직접 입력으로 모든 환경에서 동작합니다.

📋

클립보드 히스토리

최근 복사한 항목을 패널에서 빠르게 다시 붙여넣기. 반복 작업을 효율적으로 처리하세요.

🔔

자동 업데이트 알림

GitHub Releases와 연동해 새 버전이 있을 때 앱 상단 배너로 알려줍니다. 설정 없이 항상 최신 버전을 유지하세요.

Actions

키 하나에 무엇이든

단순한 키 전송을 넘어, 앱 실행·상용구 입력·볼륨 조절까지 모든 액션을 키에 연결할 수 있습니다.

⌨
SendKey
가상 키코드 단일 전송
🔗
SendCombo
키 조합 전송 (Ctrl+C 등)
📌
ToggleSticky
수식자 키 Sticky 상태 토글
🔀
SwitchLayout
다른 레이아웃으로 즉시 전환
🚀
RunApp
앱 또는 파일 실행 (경로 + 인수)
📝
Boilerplate
상용구 텍스트 유니코드 직접 입력
💻
ShellCommand
CMD / PowerShell 명령 실행
🔊
VolumeControl
볼륨 올리기·내리기·음소거
📋
ClipboardPaste
텍스트를 클립보드 복사 후 붙여넣기
Layout

JSON으로 레이아웃 정의

layouts/ 폴더의 JSON 파일 하나가 키보드 레이아웃 하나입니다. GUI 에디터로 편집하거나 직접 파일을 수정하세요. 

// layouts/my-layout.json
{
  "name": "나만의 레이아웃",
  "language": "ko",
  "rows": [
    {
      "keys": [
        {
          "label": "📋 상용구",
          "action": {
            "type": "Boilerplate",
            "text": "안녕하세요, 반갑습니다."
          },
          "width": 3.0
        },
        {
          "label": "🚀 메모장",
          "action": {
            "type": "RunApp",
            "path": "notepad.exe"
          },
          "width": 2.0
        },
        {
          "label": "🔊",
          "action": {
            "type": "VolumeControl",
            "direction": "up",
            "step": 10
          },
          "width": 1.0
        }
      ]
    }
  ]
}
