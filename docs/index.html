최적의 화상 키보드

한국어 조합 엔진(KoreanInputModule)을 내장한 Windows용 커스터마이징 화상 키보드입니다. 자동완성, TTS, 스위치 스캔은 물론, VirtualKey 홀드 입력, 상단바 더블 클릭 접기, 독립 편집 도구와 AI 텍스트 처리까지 한 흐름으로 제공합니다.

Features

필요한 기능은 모두 담았습니다

접근성부터 생산성 자동화까지, AltKey 하나로 해결하세요.

🔒

Sticky Keys

Shift·Ctrl·Alt·Win을 한 번 탭하면 일회성 고정, 두 번 탭하면 영구 잠금. 한 손 조작이나 접근성 환경에 최적화되어 있습니다.

⏱

체류 클릭 (Dwell Click)

키 위에 마우스를 올려놓으면 설정한 시간 후 자동 클릭. 물리적 클릭이 어려운 사용자를 위한 접근성 기능입니다.

🎨

JSON 레이아웃

원하는 키 배치를 JSON으로 정의하고, GUI 레이아웃 에디터로 쉽게 편집하세요. 한글 음운 이중 레이블과 가/A 토글(QuietEnglish)을 지원합니다.

Fn

Fn 레이어

키 하나에 기본 동작과 Fn 전용 동작을 함께 배치할 수 있습니다. 한 번 누르면 1회 적용, 두 번 누르면 고정되어 보조 키 공간을 아낄 수 있습니다.

🔄

앱별 레이아웃 프로필

게임, 크리에이티브 툴, 오피스 앱마다 레이아웃을 자동으로 전환합니다. WinEventHook 기반으로 포그라운드 앱을 감지합니다.

🪟

자동 투명도 UI

사용 중이 아닐 때 창 투명도를 낮춰 시야를 덜 가리고, 필요할 때는 다시 또렷하게 돌아옵니다. 장식 효과보다 입력 가독성과 접근성 유지에 초점을 맞춘 동작입니다.

🧲

화면 가장자리 이동

헤더 중앙의 ← ↑ ↓ → 버튼 한 번으로 키보드를 화면 끝에 즉시 붙입니다. 자주 쓰는 위치에 빠르게 고정하세요.

✋

VirtualKey 홀드 입력

일부 가상 키는 길게 누르는 동안 실제 키를 누르고 있는 것처럼 유지됩니다. 마우스를 떼거나 버튼 밖으로 벗어나면 즉시 안전하게 해제됩니다.

🖱

헤더 더블 클릭 접기

상단바의 빈 공간이나 드래그 핸들을 더블 클릭해 키보드를 빠르게 접고 펼칠 수 있습니다. 드래그는 일정 거리 이상 움직였을 때만 시작되어 오동작을 줄입니다.

↕

동적 행 높이

레이아웃 편집기에서 각 행을 기본 또는 낮은 높이로 조정할 수 있습니다. 숫자열과 보조줄은 더 촘촘하게, 핵심 키 줄은 더 크게 설계할 수 있습니다.

😊

이모지 패널

이모지 패널로 자주 쓰는 이모지를 빠르게 입력하세요. 유니코드 직접 입력으로 모든 환경에서 동작합니다.

📋

클립보드 히스토리

최근 복사한 항목을 패널에서 빠르게 다시 붙여넣기. 반복 작업을 효율적으로 처리하세요.

🧠

문맥 인식 자동완성

이전 단어를 기억해 다음 단어 제안의 정확도를 높이는 빅그램 학습. "안녕" 다음에 "하세요"가 먼저 올라오듯, 내 입력 패턴을 스스로 학습합니다.

🗂

독립 편집 도구

레이아웃, 사용자 단어, 프로필 매핑, AI 프롬프트를 AltKey.Tools에서 따로 편집합니다. 메인 키보드 입력을 방해하지 않고 저장 후 즉시 반영됩니다.

🏷

전체 키 강조 스타일

문자 키를 포함한 모든 키에 소프트 액센트 강조를 줄 수 있습니다. 자주 찾는 키를 더 빠르게 식별하도록 돕는 접근성 중심 옵션입니다.

✨

AI 텍스트 처리

상단바 버튼이나 키 액션으로 선택 텍스트를 요약·정리·말투 변환할 수 있습니다. 긴 한글 프롬프트는 별도 편집기에서 더 안정적으로 다룰 수 있습니다.

📚

사용자 단어 직접 관리

학습된 자동완성 단어와 관련 데이터를 편집기에서 직접 확인하고 정리할 수 있습니다. 잘못 배운 추천어를 빠르게 바로잡을 수 있습니다.

🇰🇷

한국어 조합 엔진

자체 개발한 KoreanInputModule을 통해 유니코드 기반의 완벽한 한글 조합과 정밀한 텍스트 전송을 보장합니다.

🎙️

음성 안내 (TTS)

키 라벨을 음성으로 읽어주어 시각 보조가 필요한 사용자를 돕습니다. 자모 이름 매핑을 통한 정확한 발음을 지원합니다.

⌨️

스위치 스캔 입력

외부 스위치나 지정된 키로 키보드 전체를 순차적으로 탐색. 운동 장애가 있는 사용자를 위한 핵심 보조 공학 기능입니다.

📉

애니메이션 최적화

사용자 설정에 따라 불필요한 시각 효과를 최소화하여 저사양 환경에서도 부드럽고 안정적인 입력을 지원합니다.

🔔

자동 업데이트

GitHub Releases와 연동해 새 버전이 있을 때 배너로 알려주고 클릭 한 번으로 최신 환경을 유지할 수 있게 돕습니다.

Editors

편집 작업은 별도 도구에서

AltKey.Tools는 메인 키보드와 분리된 독립 편집 창입니다. 긴 목록과 긴 한글 텍스트도 더 안정적으로 다루고, 저장 전 배치 한도 검증도 수행할 수 있습니다.

🎨
레이아웃 편집기
키 배치와 액션을 수정
📚
사용자 단어 편집기
학습된 자동완성 단어를 정리
🔄
프로필 매핑 편집기
앱별 레이아웃 자동 전환 규칙 편집
📝
AI 프롬프트 편집기
긴 한글 프롬프트를 별도 창에서 편집
🏷
상단바 단축키 편집기
커스텀 상단바 버튼의 아이콘·툴팁·액션 편집과 저장 전 한도 검사
Layout

JSON으로 레이아웃 정의

layouts/ 폴더의 JSON 파일 하나가 키보드 레이아웃 하나입니다. GUI 에디터로 편집하거나 직접 파일을 수정하세요. 

// layouts/my-layout.json
{
  "name": "나만의 레이아웃",
  "columns": [
    {
      "rows": [
        {
          "keys": [
            {
              "label": "📋 상용구",
              "action": {
                "type": "Boilerplate",
                "text": "안녕하세요, 반갑습니다."
              },
              "width": 3.0
            },
            {
              "label": "🔊",
              "action": {
                "type": "VolumeControl",
                "direction": "up"
              },
              "width": 1.0
            }
          ]
        }
      ]
    }
  ]
}
