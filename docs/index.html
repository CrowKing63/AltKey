필수 가이드 영상 매뉴얼 다운로드 피드백

Windows 10 / 11 · 한국어 전용 · AltKey.Tools 포함

한국어 입력을 더 편하게 만드는 화상 키보드

AltKey는 마우스, 터치, 스위치만으로 한글 입력과 자주 쓰는 작업을 처리할 수 있게 돕습니다. 자동완성, 체류 클릭, TTS, 외부 편집 도구를 한 화면 흐름으로 묶었습니다.

최신 버전 다운로드 매뉴얼 바로가기

Essential

처음 쓰기 전에 3가지만 확인하세요

1

설치 후 실행

설치 프로그램 또는 포터블 압축 파일을 받아 실행합니다. 편집 도구는 함께 포함됩니다.

2

자동완성 켜기

자동완성을 켜면 AltKey가 한글을 직접 조합해 보내므로 한/영 상태 오류가 줄어듭니다.

3

가/A와 한/영 구분

가/A는 AltKey 자판 전환, 한/영은 Windows IME 전환입니다.
